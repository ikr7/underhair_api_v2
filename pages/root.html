陰毛 API

陰毛っぽい画像を生成するAPIです。

PNG, JPEG, Animated GIF, SVG, JSON の出力に対応しています。

/v1/[数字]x[数字].[フォーマット] の形式でリクエストしてください。最大サイズは1024×1024(GIFのときは512×512)までに制限されています。フォーマットには png, jpg, jpeg, gif, svg, json のいずれかを指定して下さい。

パラメータ

amount
陰毛の本数を 0 - 1000 で指定します。デフォルト値は 15 です。1000 を超える値は指定できません。
curly
陰毛のチリチリ度合いを 0 - 360 で指定します。デフォルト値は 30 です。
length
陰毛のだいたいの長さを 0 - 100 で指定します。あくまでだいたいの長さです。デフォルト値は 100 です。
color
陰毛の色を CSS カラーで指定します。デフォルト値は "black" です。
bgcolor
背景色を CSS カラーで指定します。デフォルト値は "rgba(0,0,0,0)" です。

使用例

  • 320×320、PNG、完全にデフォルト
  • 320×320、PNG、本数を100本に、色を茶色に
  • 640×320、JPEG、背景を肌色に、チリチリ度弱め
  • 160×160、アニメGIF、背景を空色、色を緑に
  • 640×640、SVG、色を半透過の青に
  • 320×320、PNG、短めの陰毛を500本
  • 各パラメータで定められた範囲外の値を指定するとエラーになります
